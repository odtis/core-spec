## How to report

1. Check if the issue is already listed below.
2. File a **clarification** issue (see [Feedback channels](FEEDBACK.md)) or errata PR.
3. Editors record accepted errata here with date and affected sections.

Normative MUST changes **require** an RFC ([RFC template](RFC-TEMPLATE.md)), not errata alone.
