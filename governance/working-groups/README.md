## Proposed groups

| WG | Scope | ODTIS sections | Liaison |
|----|-------|----------------|---------|
| **Connect** | Core Identity, OIDC deltas, Verification API | 2, 3, 5 | OIDF AB/Connect, eKYC IDA |
| **Trust Network** | Gateway, TEP, grants | 4 | IETF TEP track |
| **Federation** | Bilateral operator federation | 6 | IRTF; ≠ OpenID Federation |
| **Operator** | PKI, audit, deployment | 7-10 | NIST, sector regulators |
| **Extended** | Annex D modules | D | OIDF DCP (OID4VCI/VP) |

Profile mapping: [Profile comparison](/site/PROFILES/) | IETF track: [IETF working drafts overview](../../ietf/README.md)
