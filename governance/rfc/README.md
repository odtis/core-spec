# ODTIS RFC submissions

Normative change proposals for ODTIS `0.9.x-draft` review.

**Template:** [RFC template](../RFC-TEMPLATE.md) | **Feedback:** [Feedback channels](../FEEDBACK.md) | **Governance hub:** [Project hub](../README.md)

---

## Process

1. Copy [RFC template](../RFC-TEMPLATE.md) to this folder as `YYYY-MM-DD-short-title.md`
2. Open a GitHub issue using **ODTIS RFC** template and link the file
3. **14-day** public comment period
4. Maintainers record decision in [Review Log (YAML)](../REVIEW-LOG.yaml)
5. On acceptance: update `spec/`, `registry/`, [Changelog](../../CHANGELOG.md)
