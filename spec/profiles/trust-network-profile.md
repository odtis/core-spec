# ODTIS Trust Network profile

**Profile ID:** `trust-network`

**Depends on:** `core-identity`

**ODTIS sections:** 4

---

## Purpose

Institutional exchange between operators: exchange gateway (receiver/sender), partner PKI, service catalog, grants, and federated-ready audit events.

**Depends on:** Core Identity (`core-identity`) - subjects and LoA originate from identity-core.

**Normative scope:** section 4 plus cross-section FAL documentation (`ODTIS-0106`) - **22 registry IDs**.
