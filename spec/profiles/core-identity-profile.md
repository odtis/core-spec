# ODTIS Core Identity profile

**Profile ID:** `core-identity`

**Depends on:** `reference-architecture`

**ODTIS sections:** 2, 3, 5

---

## Purpose

Reusable digital identity: IdP (OIDC), subject registry, verification API, consent, and citizen self-service. This profile is the **minimum ODTIS claim** for any production identity platform.

**Normative scope:** sections 2 (LoA), 3 (identity services), and 5 (consent and privacy) - **45 registry IDs** after exclusions below.

---

| OpenID Connect Core 1.0 | Authentication + claims |
