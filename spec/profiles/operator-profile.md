# ODTIS Operator profile

**Profile ID:** `operator`

**Depends on:** `reference-architecture`, `core-identity`

**ODTIS sections:** 7, 8, 9, 10

---

## Purpose

Platform operator obligations spanning governance, security baseline, audit retention, and **honest deployment-phase claims**. Required for national-scale operators even when Trust Network is not yet live.

**Depends on:** Core Identity (not Trust Network).

**Normative scope:** sections 7-10 - **33 registry IDs** (`ODTIS-0501`-`0533`).

## Deployment phase declaration

Conformance statements **MUST** declare:

1. deployment phase (1-4) - `ODTIS-0532`
2. active Extended sub-modules in production - `ODTIS-0532`
3. Phase 1 MUST NOT claim inactive Extended modules - `ODTIS-0533`
